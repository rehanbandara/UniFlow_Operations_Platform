# UniFlow_Operations_Platform
UniFlow is a full stack platform designed to streamline day today campus operations. It brings together facility management, room and equipment bookings, and maintenance reporting into one unified system that is easy for students, staff, and administrators to use.

---

## 🚀 Features

### **Resource Management**
Manage rooms, labs, and equipment with detailed metadata such as capacity, location, and availability.

### **Smart Booking System**
Request and manage bookings with automatic conflict detection and a clear approval workflow.

### **Maintenance Ticketing**
Report issues with descriptions, priorities, and image uploads. Track progress from open to resolved.

### **Real‑Time Notifications**
Receive instant updates when bookings are approved, tickets change status, or new comments are added.

### **Role‑Based Access**
Secure authentication using OAuth 2.0 with permissions for users, admins, and optional technician roles.

### **Clean, Modern UI**
A responsive React interface focused on clarity, speed, and ease of use.
